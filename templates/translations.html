The Asian Kung-Fu Generation translation project

Project Preamble

The Asian Kung-Fu Generation (AKFG) translation project is a personal data-visualization experiment — part language learning, part data analysis.

There are three main aspects to the project:

  1. Start with the lyrics: I write them down, highlight words I don't know, and break them apart. Here's an example.
  2. Then I run a script to pull out various features from the song; length, unique character count, etc.
  3. Finally, try to find interesting data points from there.

Overview

First, let's just look at how long it has taken me (in minutes) to translate each song so far.

Figure 1.1 — Time Spent per Song

Bar chart showing minutes spent translating each song.

Analysis

This chart shows how many minutes it took for me to translate each song. It's interesting on its own, but not super useful if we're trying to figure out whether I'm actually getting better at translating over time. Some songs are just longer than others, and naturally take more time.

Notably, if a song has more unique characters, it's probably denser or more difficult to translate. So normalizing time by dividing 'time spent' with 'unique character count' might give a better sense of how efficient I'm becoming.

Figure 1.2 — Time / Unique Character Count (UCC)

Bar chart showing translation time divided by unique character count.

Analysis

For many linguistic reasons, it's hard to have an easy "word count" in Japanese, so we use unique character count (UCC). Here we divide each length of time by the number of unique characters within the lyrics — showing how many seconds I'm spending on each character.

For now there's not much interesting stuff to look at, but more is coming soon!
